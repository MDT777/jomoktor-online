# JOMOKTOR.online — Жомоктор.онлайн

Платформа персональных детских сказок: уникальный текст + Pixar-иллюстрации → PDF-книжка.
«Жомок» по-кыргызски — «сказка». Сайт: **https://jomoktor.online**

## Что умеет

- **Персональная сказка** по анкете: имя, возраст, интересы, любимая игрушка, жанр, пожелания
- **По фото ребёнка** (опция): герой иллюстраций похож на ребёнка (Flux PuLID); фото используется только транзитом и не сохраняется
- **Двойная ИИ-безопасность текста**: захардкоженный safety-промпт + независимый ИИ-модератор (запрещённые темы, единство темы, качество)
- **ИИ-контроль иллюстраций**: Gemini Vision проверяет каждую картинку на брак (анатомия, артефакты, соответствие сцене); брак перегенерируется или отсеивается
- **PDF-книжка** (обложка + 4 иллюстрации + текст, кириллица RU/KY) собирается на сервере и уходит владельцу в Telegram на ручную проверку, после чего отправляется клиенту на почту
- **Два языка** RU/KY с автоопределением по устройству
- Заявки → Supabase + мгновенное уведомление в Telegram

## Стек

| Слой | Технология |
|---|---|
| Фронтенд | Vite + React + TypeScript + Tailwind CSS v4 + Framer Motion |
| Хостинг | Vercel (SPA rewrites в `vercel.json`) |
| Бэкенд | Supabase Edge Functions (Deno) + Postgres |
| Текст сказки | Gemini Flash (бесплатный тариф) |
| ИИ-модератор текста и картинок | Gemini Flash / Flash Vision |
| Иллюстрации | Fal.ai: Flux schnell (~$0.003/шт) · Flux PuLID (по фото) |
| PDF | pdf-lib + fontkit (шрифт DejaVu, кириллица) |
| Авторизация | Supabase Auth (email + пароль) |

## Структура

```
src/                    # фронтенд (лендинг, форма, логин, кабинет)
  components/           # секции лендинга + модалка заявки
  i18n/                 # RU/KY переводы + автоопределение языка
  lib/                  # supabase-клиент, вызовы Edge Functions
  pages/                # /login, /account
supabase/functions/
  generate-tale/        # конвейер: текст → модерация → 4 картинки → визуальный контроль → PDF → Telegram
  submit-lead/          # заявка → БД + Telegram-уведомление
```

## Секреты (в git НЕ хранятся)

Задаются в Supabase → Edge Functions → Secrets:

| Переменная | Что это |
|---|---|
| `GEMINI_API_KEY` | ключ Google AI Studio (текст + модерация + vision) |
| `FAL_KEY` | ключ fal.ai (иллюстрации) |
| `TELEGRAM_BOT_TOKEN` | токен бота уведомлений |
| `TELEGRAM_CHAT_ID` | chat id владельца |

Публичные значения (anon key, URL проекта Supabase) находятся в `src/lib/` — они рассчитаны на браузер.

## Запуск локально

```bash
npm install
npm run dev        # http://localhost:5173
npm run build      # прод-сборка в dist/
```

Деплой: `npx vercel deploy --prod`

Примечание: бинарные картинки (`public/hero.jpg`, `public/examples/*.jpg`) хранятся в локальной копии проекта и на проде; в этот репозиторий добавляются командой `git push` с локальной машины.

## Экономика (примерно)

Сказка с 4 иллюстрациями: текст ~бесплатно (Gemini free tier) + картинки ~$0.012 (schnell) или ~$0.15 (PuLID по фото). PDF и модерация — бесплатно.
